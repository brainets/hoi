hoi.metrics.GradientOinfo#

class hoi.metrics.GradientOinfo(x, y, multiplets=None, base_model=<class 'hoi.metrics.oinfo.Oinfo'>, verbose=None)[source]#

First order Gradient O-information.

The Gradient O-information is defined as the difference between the O-information with the target variable minus the O-information without the target variable :

\[\begin{split}\partial_{i}\Omega(X^{n}) &= \Omega(X^{n}) - \Omega(X^{n}_{-i}) \\ &= (2 - n)I(X_{i}; X^{n}_{-i}) + \sum_{ k=1, k\neq i}^{n} I(X_{k}; X^{n}_{-ik})\end{split}\]

Warning

  • \(\partial_{i}\Omega(X^{n}) > 0 \Rightarrow Redundancy\)

  • \(\partial_{i}\Omega(X^{n}) < 0 \Rightarrow Synergy\)
